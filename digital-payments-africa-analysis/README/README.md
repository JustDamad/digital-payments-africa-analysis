# Digital Payments & Financial Inclusion in Africa

## 📌 Overview
This project analyzes the factors influencing digital payment adoption across selected African countries, highlighting the key drivers of financial inclusion.

## 📊 Objective
To determine whether GDP or internet accessibility has a stronger impact on digital payment usage.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📁 Dataset
World Bank Data (Financial inclusion, GDP, Internet usage)

## 🔍 Key Insight
Digital payment adoption in African economies is more strongly influenced by internet accessibility than GDP levels, highlighting the importance of digital infrastructure in driving financial inclusion.
